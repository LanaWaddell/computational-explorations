# Signal vs Noise Simulation

This project explores how structured signals can be recovered from noisy environments.

## Concept

A clean signal is generated and then corrupted with random noise.  
A simple filtering method (moving average) is applied to recover the signal.

## Goals

- Understand how noise impacts systems
- Explore filtering techniques
- Measure signal clarity before and after processing

## Future Extensions

- Adaptive filters
- Frequency-based filtering
- Visualization of signal evolution

## Observer Effect Extension

This simulation includes an "observer effect" parameter, where interaction with the system introduces additional perturbation. This reflects how measurement or interaction can influence system behavior in both physical and computational contexts.
